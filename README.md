# 💸 Sub-Bud

> A clean, intuitive subscription tracking application designed to help users manage their recurring expenses and take control of their budget.

## ✨ Features

* **Dashboard Overview:** Get a quick glance at your total monthly and yearly subscription costs.
* **Manage Subscriptions:** Easily add, edit, or remove subscriptions with custom billing cycles.
* **Secure Authentication:** User login and data management powered by Firebase.
* **Responsive Design:** A seamless, mobile-friendly UI built for any device.

---

## 🚀 Tech Stack

* **Frontend:** Next.js, React
* **Backend:** Node.js, Next.js API Routes
* **Database & Auth:** Firebase (Firestore & Firebase Authentication)
* **Styling:** CSS
